# MazePlayerIA

Simple project to understand how backtracking algorithms works by creating a maze generator with a GUI.
The project uses NodeJS and the library P5.js for the GUI.

# Intallation

## Dependencies

```
npm install
```

## Build

- Development

```
npm run builDev
```

- Production

```
npm run buildProd
```

## Execution

```
npm start
```
